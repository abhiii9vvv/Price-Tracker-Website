
# 🛒 Price Tracker Website

A responsive and interactive web application that allows users to search for products and view live pricing. Built using HTML, CSS, and JavaScript, the app is designed with a clean UI and smooth user experience. Currently being upgraded to include product-specific price history graphs.

---

## 🚀 Features

- 🔍 Product Search with live pricing display
- 📱 Fully responsive design
- 📊 Upcoming: Graphs showing price trends over time
- 🌙 Light/Dark mode toggle

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Design Tools:** Figma (for UI planning)
- **Version Control:** Git & GitHub
